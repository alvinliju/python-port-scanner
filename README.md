# Port Scanner

## Overview
This Python script is a simple port scanner that allows you to check if a specific port on a given IP address is open or closed.

## Usage
1. Clone or download the repository.
2. Navigate to the directory containing the `main.py` file.
3. Run the script using Python 3:
    ```
    python3 main.py
    ```
4. Follow the prompts to enter the IP address and port number you want to scan.
5. The script will then display whether the specified port is open or closed.
